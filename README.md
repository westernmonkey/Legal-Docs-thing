# FOUNDR

AI-powered founder analysis and decision-support platform.

FOUNDR helps founders analyze business situations through a Retrieval-Augmented Generation (RAG) pipeline, transforming raw context into structured insights, highlights, and actionable recommendations.

## Features

- Founder situation analysis
- RAG-powered reasoning pipeline
- Real-time analysis visualization
- Insight extraction and highlighting
- Interactive pipeline monitoring
- Summary generation
- Modern React interface
- Dark-mode dashboard UI

---

## How It Works

1. Founder enters a business challenge or situation.
2. The system processes the input through an analysis pipeline.
3. Relevant concepts and keywords are extracted.
4. The RAG engine generates structured insights.
5. Results are visualized in real time.
6. A final summary and recommendations are produced.

## Installation

Clone the repository:

```bash
git clone https://github.com/westernmonkey/Legal-Docs-thing.git
```

Move into the project:

```bash
cd Legal-Docs-thing
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

(or the port shown in your terminal)
